## 🌟 커밋 컨벤션
|이름|내용|
|:--|:--|
|✨ feat| 새로운 기능 추가|
|🐛 fix| 버그 수정|
|📝 docs| 문서 수정|
|💡 comment| 필요한 주석 추가 및 변경|
|🎨 style| 코드 포맷팅, 세미콜론 누락, 코드 변경이 없는 경우|
|♻️ refactor| 코드 리팩토링|
|🔧 update| 코드 업데이트 및 수정|
|✅ test| 테스트 코드|
|📦 chore| 빌드 업무 수정, 패키지 매니저 수정|
|🔥 remove| 파일을 삭제하는 작업만 수행|
|🚚 rename| 파일 혹은 폴더명을 수정하거나 옮김|
|🚑 !HOTFIX| 급하게 치명적인 버그 고침|
|💥 !BREAKING CHANGE| 커다란 API 변경|
